# Bike Resale Price Analysis Using Web Scraping & EDA

An end-to-end Data Analytics project that collects real-world used bike listings from BikeKharido using web scraping and analyzes the factors affecting resale prices using Python, Exploratory Data Analysis (EDA), and Power BI.

---

## 📌 Project Overview

The objective of this project is to understand the used bike market by collecting live listings through web scraping, cleaning the data, performing exploratory data analysis, and creating an interactive dashboard to generate business insights.

This project demonstrates practical skills in:

- Web Scraping
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Business Insight Generation
- Dashboard Development

---

## 💡 Key Insights

- 📅 Newer registration years have higher resale prices.
- 🚴 Bikes with higher kilometers driven generally sell for less.
- 👤 First-owner bikes dominate the market and retain higher value.
- 🏙 Metro cities have higher average resale prices.
- 🏍 Premium brands command significantly higher resale values.
- 📉 Ownership and mileage together strongly influence depreciation.
